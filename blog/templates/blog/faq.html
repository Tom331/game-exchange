A trade represents a single entry by one user which indicates a game they own, and a game they want in exchange for the game they own. You can create one by going to the “Propose New Trade” page.

Example: Jack owns the game Grand Theft Auto 5 and is willing to trade that for Far Cry 5. Jack goes to the “Propose New Trade” page and enters Grand Theft Auto 5 in the “Game you own” field, and Far Cry 5 in the “Game you want in exchange” field.
When one user submits a trade, they indicate the game they own and the game they want in exchange. If another user submits a trade, it is considered a Match if the game the first user owns is the game the second user wants, AND the game the first user wants is the game the other second user owns.

Example: Jack creates a new Trade. He owns Grand Theft Auto 5 and wants Far Cry 5 in exchange. Jill creates a new trade. She owns Far Cry 5 and wants Grand Theft Auto 5 in exchange. This would be considered a Match, and would show on BOTH Jack and Jill’s “Matches” page.
A Confirmed Trade is when one of the two users (you or the other user) from a Match has shown interest in following through with the trade. If a user opens their Matches page, a list will be displayed if any of the trades they submitted matches with any trades by other users.

A “Confirm Trade” button is available on each match. If either user clicks “Confirm Trade”, it will be added to their “Confirmed Trades” page. You can open a Confirmed Trade from that page by clicking the “Open trade” button. There you will see the trade’s current status, the username of the user you matched with, the two games involved in the trade and a chat window where you can arrange to meet the other user.
Waiting for 2nd confirmation from [username]: One user of a match has clicked “Confirm Trade”. The other user from the Match has not confirmed the trade yet.

Open: Both users of a Match have clicked “Confirm trade” and both plan on going following through with the trade. A chat window is now open for the users to arrange a meeting.

Cancelled: Manually cancelled by [username]: One of the users has cancelled the trade and no longer wishes to follow through.

Cancelled: Auto-cancelled because [username] did not confirm the trade within 2 days: One user clicked “Confirm trade” on a Match, but the other user did not confirm within three days, so the trade was cancelled.

Auto-cancelled due to inaction: Both users from a Match confirmed a trade, but did not complete the trade within 9 days of the first user clicking “Confirm trade”
